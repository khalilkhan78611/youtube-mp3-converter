About This Tool

YouTube to MP3 Downloader

This is a simple web application built with Python and Flask that allows you to extract audio from YouTube videos and download them as MP3 files.

Technologies Used

  • Python
  • Flask (Web Framework)
  • pytube (YouTube downloading library)
  • pydub (Audio processing)
